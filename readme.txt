== Description ==

The most powerful quiz plugin for WordPress. Highly extendable and developer-friendly. Requires ACF Pro. Forked in 2016 from WP Pro Quiz.

= Features =
* Single Choice question type
* Multiple Choice question type
* Sorting Choice question type
* Free Choice question type
* Fill in the Blank question type
* Time limit for quiz taking
* Randomize questions & answers
* Correct or incorrect response message for all questions
* Different points for each answer
* Result text with gradations
* Statistics
* Quiz requirements
* Hints
* Email notifications
* Categorization and tagging support for both quizzes and questions

== Installation ==

1. Upload the "quizmaster" folder to the /wp-content/plugins/ directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Visit the "QuizMaster" admin menu item to begin adding quizzes and questions
